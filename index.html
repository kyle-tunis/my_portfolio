Latest Project

My first JavaScript project was conducted through the codecademy full-stack bootcamp. The challenge asked me to create a program that would take 3 arrays of random strings and produce a completely random output. When I was told to do this, I couldn't help but think of that silly game I would place in elementary school with my friends called MASH. The idea of mash would be, you would take a random list of places you could reside, careers, salary, # of kids etc. And when you play the game it would give you a randomization of results. For example (Your spouse is Taylor Swift, your job is an Astronaut, you will have 4 kids and drive a Beatle.). For now this program is strictly in Javascript. If you would like to try it, go to the following link: MashBot! Download the script.js file and give it a shot!
